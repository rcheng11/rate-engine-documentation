Animation Configuration

Specify the spritesheet grid positions as pairs with the following keys in the mapFrames value:

  • "w", "a", "s", "d"
  • "l", "r", "t", "b" for ->, <-, ↑, ↓ keys respectively
  • "i" for idle animation (can be a single frame)
{
"filepath": "./assets/sprites/spriteName.bmp",
"fps" : 12,
"width": w,
"height": h,
"frameW": A,
"frameH": B,
"mapFrames": {
"i": {
"framesArray": [0, 1, 2, 3],
"flipType": "X"
},
"w": {
"framesArray": [4, 5, 6, 7],
"flipType": "X"
}
}
}

JSON File Explanation

  • w: Width of spritesheet
  • h: Height of the spritesheet
  • A: Width of a single animation frame in the spritesheet
  • B: Height of a single animation frame in the spritesheet
  • mapFrames Array: Represents an animation sequence mapped to a key
  • framesArray: Spritesheet sliced into a grid, each number represents a grid position. See "Frames Array Explanation".
  • flipType: Can be one of "FR", "FL", or "X":
    • "FR": Animation sequence is flipped horizontally when the sprite moves right.
    • "FL": Animation sequence is flipped horizontally when the sprite moves left.
    • "X": Animation sequence is not flipped.
