Detailed Description

Defines a BEHAVIOR component that operates logic on the game's data/icomponents

  • Movement
  • Collision
  • Click

These script components should ALWAYS contain a reference to the game object it's attached to! This is because the purpose of a script component is to MODIFY a particular game object.
