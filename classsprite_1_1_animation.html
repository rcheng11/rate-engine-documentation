Detailed Description

Optional added component to Sprite, holds FrameArray objects mapped to inputs.

  • Not a game component, but an optional addition to the sprite
  • This will modify the selection rectangle of the sprite at approx fps speed, assumes game runs at approx. 60 fps
  • Uses a framesMap of framesArrays for both idle and mapped animation
