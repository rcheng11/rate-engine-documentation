Detailed Description

flip type affects the object horizontally and is one of the following: "FL"--> Performs a flip when the sprite is "facing" left "FR" --> Performs a flip when the sprite is "facing" right "FA" --> Always performs a horizontal flip "X" --> The object is not flipped
