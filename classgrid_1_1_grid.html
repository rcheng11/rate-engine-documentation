Detailed Description

The Grid object represents a zero-indexed, uniform grid on the screen (see below 4x4 grid):

0 1 2 3
0 (0,0) (0,1) (0,2) (0,3)
1 (1,0) (1,1) (1,2) (1,3)
2 (2,0) (2,1) (2,2) (2,3)
3 (3,0) (3,1) (3,2) (3,3)

Its main functions are:

  • Supports GUI by making it easier to position objects
  • Renders a grid on the screen
  • Translates a position to the corresponding centered grid position.

Overloaded constructor for Grid, will create a grid based on squares with an area of unitArea (float). Will complain if this grid is not able to fit into the grid.

Defaults to 4096 pixels, which is the size of a 64x64 sprite, which are the sizes used in the assets folder.
